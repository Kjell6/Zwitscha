Hallo meine Webbis!

Wenn Sie eine URL ohne Dateinamen aufrufen, wird standardmäßig (falls vorhanden) entweder die Datei index.html oder index.php ausgeliefert. Wenn Sie im Laufe des Semesters eine Webanwendung in PHP programmieren, müssen Sie diese statische HTML-Datei entsprechend durch eine index.php-Datei ersetzen.
